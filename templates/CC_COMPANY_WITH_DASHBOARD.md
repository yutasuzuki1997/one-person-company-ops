# 1人会社 + AI Agents ダッシュボード（貼り付け用メモ）

このリポジトリでは **cc-company** で `.company/` を育てつつ、**AI Agents ダッシュボード** で同じ「社員」たちの稼働を一覧しています。

- Claude Code: `/company` → 秘書・TODO・部署
- ダッシュボード: 各メンバーへの一斉指示・個別トーク・稼働表示

プロジェクトの **repoPath** をこのディレクトリに合わせると、ダッシュボード上部に **今日の秘書TODO** が表示されます。

詳細: リポジトリ内 `docs/ONE_PERSON_COMPANY.md`
