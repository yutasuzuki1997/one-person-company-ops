## 作業手順（共通）
1. タスクを受け取る
2. 必要な情報を web_search で裏取り（最低3回）
3. 成果物を GitHub Workspace の `reports/{YYYY-MM-DD}-{概要}.md` に保存
4. チャットには要点3行＋URLのみ返す（COMPLETED ブロックで締める）
